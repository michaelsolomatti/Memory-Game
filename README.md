🧠 Memory Game – Найди пару на 7 языках
Классическая игра на память в терминале!
Открывай карточки, находи пары, тренируй мозг и улучшай свою память.
Реализована на семи языках программирования с единым интерфейсом, таймером, счётчиком ходов, таблицей рекордов и настраиваемой сложностью.

🚀 Возможности
Игровое поле – сетка из карточек (4×4, 4×6, 6×6 и т.д.) с парами одинаковых символов.

Открытие карточек – игрок выбирает две карточки по координатам; если они совпадают – остаются открытыми.

Счётчик ходов – количество попыток для завершения игры.

Таймер – засекает общее время игры.

Рекорды – сохраняются лучшие результаты (количество ходов и время) для каждого уровня сложности.

Уровни сложности – выбор размера поля (маленький, средний, большой).

Цветной вывод – карточки подсвечиваются разными цветами для лучшей визуализации.

Интерактивность – ввод координат с клавиатуры, отображение текущего состояния поля.

Выход – в любой момент можно выйти, нажав q или Ctrl+C.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --size <N>	Размер поля (N×N), например 4, 6, 8 (по умолчанию 4)
-l, --level <easy|medium|hard>	Уровень сложности (переопределяет размер): easy=4×4, medium=6×6, hard=8×8
-t, --timeout <сек>	Таймаут на ход (если превышен, карточки закрываются)
-h, --help	Справка
Примеры
bash
# Запустить игру с полем 4×4
python memory.py

# Игра с полем 6×6
python memory.py -s 6

# Уровень сложности hard (8×8)
python memory.py -l hard

# Таймаут 5 секунд на ход
python memory.py -t 5
Управление
Введите координаты карточки в формате строка столбец (например, 1 2).

q – выход.

После игры будет показан результат и предложено начать заново.

🧠 Логика игры
Генерация поля – создаётся матрица N×N, заполненная парами случайных символов (буквы, цифры или эмодзи).

Перемешивание – порядок карточек случайным образом перетасовывается.

Отображение – все карточки изначально скрыты (показаны как ■ или ?).

Ход игрока – игрок вводит координаты первой карточки, она открывается, затем вводит координаты второй.

Проверка – если карточки совпадают, они остаются открытыми; если нет – закрываются через небольшую задержку.

Завершение – игра заканчивается, когда все пары найдены.

Результат – вывод количества ходов, времени, обновление рекорда.

✨ Дополнительные фичи
Анимация – карточки открываются/закрываются с паузой.

Цветные символы – каждый символ имеет свой цвет для лучшего запоминания.

Рекорды – сохраняются для каждого размера поля в файл ~/.memory_records.json.

Подсказка – при таймауте карточки автоматически закрываются.

Статистика – вывод средней скорости на ход.
